# bitwise operators in C

> all operators in C
>
> * && logial AND
> * || logical OR
> * ! logical NOT
> * ^ logical XOR
> * & bitwise AND
> * | bitwise OR
> * ^ bitwise XOR

> * << bitwise left shift
> * \>> bitwise right shift
> * ++ increment + assign
> * -- decrement + assign
> * += increment + assigment
> * -= decrement + assigment
> * *= multiplication
> * /= division
> * %= modulus
